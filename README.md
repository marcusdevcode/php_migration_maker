# How it's works?
1. Clone the repo 
2. Search and replace `SampleNameSpace` with your namespace that you use in your workflow.
3. Open `mm` file in order to change Migrations folder location.
4. Run command `php mm` it will generate file using 'newmigrationtemplate.subs' file as template

You can redefine `MMCLASSESPATH` in order to change Migration folder path. 

You can redefine `MMFILE` in order to change Migration class instance.
